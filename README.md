# Disaster Relief Resource Locator

A web application to help locate essential resources during disasters.

## Features

- Interactive map powered by Google Maps API
- Search for resources by name, type, or location
- Use current location to find nearby resources
- Display resource details including type and address

## Setup

1. Get a Google Maps API key from the [Google Cloud Console](https://console.cloud.google.com/)
2. Replace `YOUR_API_KEY_HERE` in `index.html` with your actual API key
3. Open `index.html` in a web browser

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Google Maps JavaScript API

## Sample Data

The application includes sample disaster relief resources for demonstration purposes. In a real-world scenario, this data would be fetched from a backend API or database.
